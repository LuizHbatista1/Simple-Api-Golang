# Basic Go API

This project is a basic API implemented in Golang with three primary methods: GET, POST, and DELETE. The API is designed to perform basic CRUD (Create, Read, Update, Delete) operations.

## Requirements

Go 1.16 or later

## API Endpoints

GET /items
Fetch all items.


POST /items
Create a new item.

DELETE /items/{id}
Delete an item by ID.
